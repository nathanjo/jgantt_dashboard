jGantt Content Dashboard is a simple Calendar content tracking.
As administrator some tools you can use:

<ul>
  <li>Add what content type to track.</li>
  <li>Ability to select date field.</li>
  <li>Color of choice for easy category.</li>
  <li>Nodequeue integration.</li>
</ul>

<h2>Dependencies:</h2>

<a href="https://drupal.org/project/libraries">Libraries API</a> - 2.x

<h2>Installation</h2>

1. Download the latest version of the <a href="https://github.com/taitems/jQuery.Gantt">jQuery.Gantt</a> and extract it to sites/all/libraries or sites/sitename/libraries as you require.
2. Enable jGantt Content Dashboard
3. Configure content types through Dashboad settings in admin/config/jgantt_dashboard.

<h2>Related Modules</h2>
